# EvoKnight

EvoKnight is an action-platformer game mainly inspired by Celeste and Hollow Knight.



## Context

2-weeks challenge I do with friends to learn Godot and make an action-platform game.

## Game features

- Stats
	- HP
	- Strength
	- Defense
	- EXP
	- Mana

- Abilities
	- Double Jump
	- Basic Slash (4-directions)
	- Magic Slash (Left/Right)
	- Dash (8-directions)

- Enemies
	- Bat
	- Grounded spiky enemy
	- Flying spirit
	- Flying spiky enemy
	- Bigger bat with minions
	- Boss with similar abilities

## Controls

### Keyboard (QWERTY)

- `WASD`: Movement
- `SPACE`: Jump/Confirm
- `I`: Confirm
- `P`: Pause
- `J`: Basic Slash
- `K`: Dash
- `L`: Magic Slash

### Controller (XBOX)

- `Left Joystick`: Movement
- `Button A`: Jump/Confirm
- `Button B`: Confirm
- `Start`: Pause
- `Button X`: Basic Slash
- `Right Trigger`: Dash
- `Left Trigger`: Magic Slash
