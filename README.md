# Godot Game Jam

## Context

2-weeks challenge I do with friends to learn Godot and make an action-platform game.

## Game features

- Stats
	- [x] HP
	- [x] Strength
	- [x] EXP
	- [x] Mana

- Abilities
	- [x] Double Jump
	- [x] Basic Slash (4-directions)
	- [ ] Dash
	- [ ] Magic Slash (Left/Right)

- Enemies
	- [x] Bat
	- [ ] Flying mage
	- [ ] Grounded spiky enemy
	- [ ] Boss with similar abilities

## Controls

- `Left/Right`: Movement
- `SPACE`: Jump
- `Q/A`: Basic Slash
- `W/Z`: Dash
- `E`: Magic Slash
